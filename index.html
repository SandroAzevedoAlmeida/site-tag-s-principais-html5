Outras tags

A tag "p" simplesmente adiciona um parágrafo a tudo que estiver dentro dela.

A tag "mark" adiciona a cor amarela à palavra, indicando que é algo importante no texto.

A tag "small" é utilizada para identificar um trecho menor no texto, indicando pouca importância. Também usado para notas de rodapé.

A tag "i" é utilizada para fazer o texto aparecer em itálicoSS.

A tag "u" é para formatação de texto: ela sublinha o trecho escolhido.

A tag "strong" faz o trecho ficar em negrito.

A tag "ol" serve para enumerar uma lista, vem seguida da tag "li", que serve para marcar as mesmas. Exemplo:

  1. -Primeiro item da lista;
  2. -Segundo item da lista;
  3. -Terceiro item da lista;
  4. -Quarto item da lista;

A tag "ul" serve para criar uma lista não ordenada, onde não há a necessidade de enumerá-la. Também vem seguida da tag "li":

  • -Primeiro item da lista;
  • -Segundo item da lista;
  • -Terceiro item da lista;
  • -Quarto item da lista;

A tag "a href" serve para criar links entre as páginas da web.

A tag "sub" serve para indicar textos subscritos.

A tag "blockquote"

é usada para destacar ou citar blocos de texto longos que foram extraídos de outra fonte.

A tag "font" determina o tamanho, cor e tipo de fonte,como neste exemplo.
